# ⚠️ Exception Handling in Java

Exception Handling is a mechanism that allows Java programs to detect, handle, and recover from runtime errors without crashing the application.

Instead of terminating unexpectedly, Java provides tools to manage errors gracefully and improve application reliability.

# 🧠 Key Takeaways

- Exceptions prevent applications from crashing unexpectedly.
- try contains risky code.
- catch handles exceptions.
- finally executes whether an exception occurs or not.
- throw creates an exception.
- throws declares that a method may throw an exception.
