# JDK, JRE and JVM

## What I Revised
- What is Java?
- Difference between JDK, JRE and JVM
- How Java achieves platform independence

## Key Points
- JDK = Stands for Java Development Kit Development tools + JRE
- JRE = JRE stands for Java Runtime Environment and it's combined term used for JVM + Libraries required to run Java programs
- JVM = Stands for Java Virtual Machine ,Executes bytecode and makes Java platform independent

## Real World Understanding
Write Code (.java)
        ↓
Compile
        ↓
Bytecode (.class)
        ↓
JVM
        ↓
Program Runs
