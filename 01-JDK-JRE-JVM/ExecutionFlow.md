# Java Execution Process

```text
HelloWorld.java
      ↓
javac (Compiler)
      ↓
HelloWorld.class (Bytecode)
      ↓
JVM
      ↓
Machine Code
      ↓
Program Output
```

## Summary

- **JDK** is used to develop Java applications.
- **JRE** provides the runtime environment.
- **JVM** executes the bytecode and makes Java platform-independent.
