# ☕ Classes & Objects in Java

Classes and Objects are the heart of Object-Oriented Programming (OOP). Almost every Java application is built using classes and objects.

Instead of writing everything inside one large program, Java encourages us to model real-world entities such as Students, Cars, Employees, or Bank Accounts as objects.

# 🧠 Key Takeaways

- A Class is a blueprint.
- An Object is a real instance of that blueprint.
- One class can create many objects.
- Every object has its own data but shares the same structure.
- OOP helps make software reusable and organized.
