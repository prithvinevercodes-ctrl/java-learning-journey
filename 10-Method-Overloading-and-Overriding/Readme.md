# ☕ Method Overloading & Method Overriding in Java

Polymorphism allows the same method name to behave differently depending on the situation.

Method Overloading provides multiple ways to perform similar tasks using the same method name, while Method Overriding allows child classes to provide their own implementation of a parent class method.

# 🧠 Key Takeaways

- Overloading = Same method, different parameters.
- Overriding = Child class replaces parent behavior.
- Overloading happens at compile time.
- Overriding happens at runtime.
- Polymorphism makes software flexible and extensible.
