## 💡 Why Conditional Statements Matter

Conditional statements allow a program to make decisions based on specific conditions.

They are used in almost every software application.

Examples:
- Login authentication
- ATM transaction validation
- Student grade calculation
- Traffic signal systems
- Shopping cart discounts
- Banking applications

## 🧠 Key Takeaways

- `if` executes a block only when the condition is true.
- `if-else` helps choose between two possibilities.
- `else-if` is useful when multiple conditions need to be checked.
- Nested `if` allows one condition inside another.
- `switch` improves readability when checking multiple fixed values.

---

## 💭 Reflection

Conditional statements are where Java programs start behaving intelligently.

Every real-world application—from login pages to banking systems—depends on decision-making logic. Understanding these concepts makes programming feel much more practical than just writing syntax.
