# 🎭 Polymorphism in Java

Polymorphism is one of the four pillars of Object-Oriented Programming (OOP). The word "Polymorphism" means **"many forms."**

In Java, it allows the same method or interface to perform different actions depending on the object that invokes it.

Polymorphism makes applications more flexible, reusable, and easier to extend.

# 🧠 Key Takeaways

- Polymorphism means "one interface, many implementations."
- Compile-time polymorphism is achieved using method overloading.
- Runtime polymorphism is achieved using method overriding.
- Dynamic Method Dispatch decides which overridden method executes at runtime.
- Polymorphism improves flexibility and scalability.
