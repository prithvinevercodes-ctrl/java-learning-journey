# 🧬 Inheritance in Java

Inheritance is one of the four pillars of Object-Oriented Programming (OOP). It allows one class to inherit the properties and behaviors of another class, promoting code reusability and a hierarchical structure.

Instead of writing the same code multiple times, Java enables child classes to reuse and extend the functionality of parent classes.

# 🧠 Key Takeaways

- Inheritance promotes code reusability.
- A child class inherits fields and methods from its parent.
- Java uses the `extends` keyword for inheritance.
- Inheritance represents an **IS-A** relationship.
- Child classes can add their own properties and methods.
- `super` allows access to parent class members.
