# 🔒 Encapsulation in Java

Encapsulation is one of the four pillars of Object-Oriented Programming (OOP).

It combines data and methods into a single unit while restricting direct access to the object's data. This is achieved using private variables and public getter and setter methods.

# 🧠 Key Takeaways

- Keep variables private.
- Access data using getters and setters.
- Protect object data from invalid updates.
- Encapsulation improves maintainability and security.
- Validation can be performed inside setter methods.
