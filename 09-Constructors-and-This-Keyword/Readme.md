# ☕ Constructors & this Keyword in Java

Constructors are special methods that initialize objects when they are created.

Instead of assigning values manually every time, constructors allow us to create fully initialized objects from the beginning.

The `this` keyword helps Java distinguish between instance variables and local variables while also supporting constructor chaining.

# 🧠 Key Takeaways

- Constructors initialize objects.
- Constructors have the same name as the class.
- Constructors do not have a return type.
- Constructor overloading allows multiple ways to create an object.
- `this` refers to the current object.
- `this()` helps reuse constructor logic.
